# MagiDash Infrastructure

## Prerequisites

1. [terraform](https://www.terraform.io/downloads.html)
1. [terragrunt](https://terragrunt.gruntwork.io/docs/getting-started/install/)

## Setup

```sh
$ git clone git@github.com:ChrisWilding/magidash.git
$ cd infrastructure
```

## How To

### Apply

```sh
$ terragrunt init
$ terragrunt apply
```
